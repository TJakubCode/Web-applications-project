# Technologies used
- react
- sqlite
- express
- jsonwebtoken
- dotenv
- bcrypt
# Source of data
- https://fakestoreapi.com/products
# Setup
## Server:

### Create .env
JWT_SECRET=[YOUR KEY] JWT_EXPIRES_IN=1h

### Running:
To run the server:
```shell
cd backend
npm install
npx tsx server.ts
```

## Website
```shell
cd frontend
npm install
npm run dev
```
